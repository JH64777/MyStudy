함수 : 코드에 대한 이름표
변수 : 값에 대한 이름표

def hello():
    print("Hello")

만일 위와 같이 함수를 선언하면 메모리에 함수 객체가 할당이 되며 hello라는 이름이 바인딩 된다.

id 함수를 통해서 함수가 객체가 할당된 주소를 확인할 수 있음
id(hello)

함수를 위한 공간을 메모리상에 할당하고 이름을 붙여준 것으로 생각하면 될 듯

다른 변수로 함수 객체를 바인딩 할 수 있음

test = hello # hello가 바인딩한 함수 객체를 test도 같이 바인딩 함
test() # test를 통해서 함수를 호출할 수도 있음

-----------------------------------
class Func:
    def __call__(self):
        print("호출 됨")

__call__ 메소드는 호출 메소드이며 객체를 생성한 후 ( )만 붙여주면 자동으로 __call__메소드가 호출된다.

test = Func()
test()
# 결과 : "호출 됨"
